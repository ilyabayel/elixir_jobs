## Exercise 03
### Start with:
```
  mix run --no-halt
```

### API
- ``/`` - Welcome page
- ``/jobs`` - Jobs page with query params requred
  - Query Params:
    - ``latitude`` [deg] any number
    - ``longitude`` [deg] any number
    - ``radius`` [km] must be > 0
  - Example: ``/jobs&latitude=0&longitude=0&radius=1``
